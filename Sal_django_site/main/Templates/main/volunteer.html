Imagine if you always knew where to find free food.

This is what we were thinking when we, Justo and Wisdom, first developed Freefare. It first started out as a project under Code For America's Connecticut branch/"brigade", Code for CT. Wisdom was working on helping a local non-profit with their website redesign, and Justo was working on the design of something called 'Sal', which was short for 'Salvage', which Justo intended to make name of the app. The initial idea was the same as it is now: give people the ability to know about where to find free food in their area. 'Sal' was supposed Of course Justo procrastinated Sal, and somewhere along the way it turned into a foodwaste project (almost same thing) and the name changed to Sal Hates Waste.
